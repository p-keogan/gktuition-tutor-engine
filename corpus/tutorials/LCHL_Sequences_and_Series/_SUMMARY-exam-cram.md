## ЁЯУЪ Learning work тАФ what must be in your head before you sit down

These are the items page 22 **doesn't** give you. Drill until they're automatic.

### 1. The `|r| < 1` convergence condition for `SтИЮ`

Printed next to the formula, but easy to miss in exam pressure. **Apply `SтИЮ` only when `|r| < 1`.** If `|r| тЙе 1` the series diverges and the formula gives nonsense.

> Paul (sequences-series-8 [04:38]): *"Beside the sum to infinity formula, it tells you that you can only use this formula if the absolute value of r is less than 1. Mathematically another way of saying that is saying that r needs to be between 1 and minus 1."*

This is also the rejection criterion when "find `r`" produces two candidates тАФ drop the one with `|r| тЙе 1`. Paul (S&S-8 [20:18]): *"only one of these answers is validтАж It's because there's a sum to infinity. If a sum to infinity exists that means that the absolute value of r has to be less than 1."*

### 2. The differences-method for quadratic and cubic patterns

Not on the tables. You must have them cold:

```
Quadratic pattern  TтВЩ = an┬▓ + bn + c   тЗФ   2nd difference is constant, equal to  2a
Cubic pattern      TтВЩ = an┬│ + bn┬▓ + cn + d   тЗФ   3rd difference is constant, equal to  6a
```

Once you have `a` from the constant difference, sub `n = 1, 2` (and `n = 3` for cubic) into the form to get simultaneous equations in the remaining coefficients.

> Paul (Patterns 1 [00:12]): *"In order to identify something as a quadratic pattern you need to prove that the second difference is common."* Also [01:21]: *"In the Leaving Cert if you've got a question like this you should mention the phrase 'second difference'."* тАФ state the test by name.

### 3. The `TтВЩ = SтВЩ тИТ SтВЩтВЛтВБ` identity

When the question gives you `SтВЩ` (not `TтВЩ`) and asks for individual terms, you can always recover `TтВЩ = SтВЩ тИТ SтВЩтВЛтВБ`. Companion identity for the slice-sum trick: `S_m тИТ S_k` = sum from term `k+1` to `m` (sequences-series-3).

### 4. The `(aтИТd, a, a+d)` and `(a/r, a, ar)` tricks

When "three consecutive terms of an AP/GP" is set up symmetrically as `(aтИТd, a, a+d)` (AP) or `(a/r, a, ar)` (GP), the middle-term parameter `a` drops out of any sum or product condition, halving the algebra (sequences-series-4 closing block).

### 5. Year-0 vs year-1 indexing in financial sequences

The `TтВЩ = a┬╖r^(nтИТ1)` formula assumes term 1 = `a`. For year-0-indexed sequences ("price in Year 0 = `a`"), use `TтВЦ = a┬╖rс╡П` тАФ raise to the power that **counts the years**. Tested 2017, 2020, 2022 DF, 2024.

## ЁЯЪи LOAD-BEARING тАФ the five things that win or lose the S&S question

### 1. The page-22 formulas, applied with the right `a`, `r`, `d`, `n`

> The formulas are given. **The marks are for setting up `a`, `r`, and `n` correctly.**

Most lost marks here aren't formula failures тАФ they're parameter-identification failures. For arithmetic: `a` = first term, `d` = (any term) тИТ (previous term). For geometric: `a` = first term, `r = TтВВ/TтВБ` (must equal `TтВГ/TтВВ` тАФ if not, it isn't geometric).

> ЁЯЪи **Tested every year. 11/11 since 2015.**

### 2. `|r| < 1` is mandatory before applying `SтИЮ`

The single most-deducted error on `SтИЮ` questions. **State the convergence check explicitly** тАФ even if it's "obviously" satisfied.

| Condition | What to do |
|---|---|
| `\|r\| < 1` | `SтИЮ = a/(1тИТr)` applies тАФ finite |
| `\|r\| тЙе 1` | Diverges. `SтИЮ` does NOT exist. Reject this `r` candidate if the question asks for `SтИЮ` |
| Two candidate `r`s, one with `\|r\| < 1` and one with `\|r\| тЙе 1` | Reject the divergent one. (2018 Q4 was exactly this тАФ pick `x = 8` over `x = 1`) |

> ЁЯЪи Tested 2015, 2016, 2017, 2018, 2019, 2021, 2022, 2024 DF тАФ every other year, on average.

### 3. "Smallest `n` for which `TтВЩ < bound`" тАФ take logs, watch the sign-flip

The geometric-decay pattern. For `0 < r < 1`:

1. Write `a┬╖r^(nтИТ1) < bound`.
2. **Take logs of both sides** (IL-5 тАФ load-bearing toolkit cross-strand).
3. **Divide by `log(r)`**, which is **negative** since `r < 1` тАФ **inequality flips**.
4. Round **up** to the next integer (smallest `n` strictly greater than the algebraic answer).

> ЁЯЪи Paul (sequences-series-8 [09:30]): *"ignore the inequality and ... actually work out what is the value for n for which TтВЩ is equal to 0.01. When I get that value at the end I can then deduce whether I need to round up or round down."* Tested 2017, 2021, 2024 DF, 2025 DF.

### 4. `SтВЩ > bound` for an arithmetic series тЗТ quadratic inequality

For arithmetic, `SтВЩ = (n/2)[2a + (nтИТ1)d]` is **quadratic in `n`**. When asked "for what `n` does `SтВЩ > K`?", set up `SтВЩ = K`, expand to `An┬▓ + Bn тИТ K = 0`, solve via the quadratic formula (page 20), take the positive root, round up.

> ЁЯЪи 2025 P1 was this exact pattern. Tested heavily in the 2020/2025 algebraтАУS&S crossover. Algebra-9 (quadratic formula) is the load-bearing partner.

### 5. Limits as `n тЖТ тИЮ` тАФ divide every term by the highest power of `n`

Calculation Type 2 from F&G-10 / S&S-14. For `lim_{nтЖТтИЮ} (3n┬▓ + 5)/(2n┬▓ тИТ n + 1)`:

```
divide top and bottom by n┬▓:   (3 + 5/n┬▓) / (2 тИТ 1/n + 1/n┬▓)  тЖТ  3/2  as  n тЖТ тИЮ
```

Each `(constant)/n^k` term tends to 0. Companion fact: `rтБ┐ тЖТ 0` whenever `|r| < 1` (which is also the `SтИЮ` convergence condition тАФ same idea).

> ЁЯЪи Tested 2020, 2022, 2023, 2023 DF, 2024. Limits is the silent half of "sequences and series and limits" тАФ easy marks if you've seen the recipe.

## ЁЯОп The 7 techniques you must execute without thinking

1. **Identify `a` and `r` (or `a` and `d`)** from the first two or three terms. For geometric, always verify `TтВВ/TтВБ = TтВГ/TтВВ` тАФ if it doesn't, the sequence isn't geometric.
2. **Prove a sequence is arithmetic / geometric** by Method 1 (show the constant-difference / constant-ratio relation directly) or Method 2 (derive `TтВЩ`, then compute `TтВЩтВКтВБ тИТ TтВЩ` or `TтВЩтВКтВБ/TтВЩ` and show it doesn't depend on `n`).
3. **"Find `a` and `r`" simultaneous equations.** Given two equations like `TтВЪ = X` and `Tq = Y`, divide them тАФ `a` cancels тАФ solve for `r`, then back-substitute (sequences-series-7).
4. **Slice-sum trick.** `S_m тИТ S_k` = sum of terms from `T_{k+1}` to `T_m`. Useful when the question asks for a partial range, e.g. "sum of terms 10 to 20".
5. **Patterns method (quadratic / cubic).** Build a difference table. Constant 2nd diff тЗТ quadratic; constant 3rd diff тЗТ cubic. Use `2a = constant 2nd diff` (or `6a = constant 3rd diff`), then 2 (or 3) simultaneous equations from `TтВБ, TтВВ` (and `TтВГ`) to get the remaining coefficients.
6. **Recurring decimal тЖТ fraction via `SтИЮ`.** Write the recurring block as a geometric series with first term = block/10^(block length) and ratio = 1/10^(block length); apply `SтИЮ` (sequences-series-9).
7. **Induction proof of `SтВЩ`.** The 4-step framework тАФ `P(1)`, assume `P(k)`, prove `P(k+1)`, conclude тАФ applied to the arithmetic or geometric sum formula (sequences-series-10). Requires the full Induction ┬з1тАУ6 chapter as prerequisite.

## тЪа Common traps тАФ where students lose marks

| Trap | Fix |
|---|---|
| Applying `SтИЮ` without checking `\|r\| < 1` | State the check explicitly. If `\|r\| тЙе 1`, the series diverges тАФ answer "does not exist" |
| Picking the wrong `r` when two candidates exist (e.g. `r = 1/2` vs `r = 2`) | If the question mentions `SтИЮ`, pick `\|r\| < 1`. That's the rejection criterion |
| Forgetting to flip the inequality when dividing by `log(r)` with `r < 1` | `log(0.9) < 0` тАФ dividing by a negative flips the inequality |
| `TтВЩ` vs `SтВЩ`: using `TтВЩ = a + (nтИТ1)d` when the question asks for the sum | Read carefully. "The 10th term" тЖТ `TтВЩ`. "The sum of the first 10 terms" тЖТ `SтВЩ` |
| Year-indexing off-by-one (5 years compounding тЙа `TтВЕ`) | Write out Year 0, Year 1, Year 2тАж in a table before applying `TтВЩ` |
| Patterns: mistaking the constant 2nd difference for `a` itself | The constant 2nd difference equals **2a**, not `a`. Halve it |
| Solving `SтВЩ > K` and rounding down | The smallest `n` for which `SтВЩ > K` is the ceiling of the algebraic answer тАФ round **up** |
| `SтВЩ` formula direction: using `(rтБ┐ тИТ 1)/(r тИТ 1)` when `r < 1` | Both forms are equivalent. The `(1 тИТ rтБ┐)/(1 тИТ r)` form keeps numerator positive when `r < 1` |
| Mixing AP and GP formulas (they sit next to each other on p. 22) | Read the formula line carefully тАФ different `T` and `S` for each |
| For `SтИЮ` on alternating series, treating `r = тИТ1/3` as divergent | `\|r\|` = `1/3` < 1, so it converges. The sign of `r` doesn't affect convergence, only the magnitude |

## ЁЯТб Three exam-day tips that move the needle

1. **State the convergence check before applying `SтИЮ`.** Two lines: "`|r| = тАж < 1`, so `SтИЮ` exists" then the formula. The check is worth a method mark every time and prevents you from applying the formula to a divergent series.

2. **Build a table for indexed problems.** Whenever the question mentions Year 0 / Year 1 / month 1 / repayment 1, write the first three rows of the table before reaching for `TтВЩ`. The off-by-one error vanishes once the table is in front of you.

3. **For Section B Q7 financial-maths, set up `a`, `r`, and `n` first тАФ then read the question again.** Q7 is dense with context; the maths only starts after you've identified the geometric structure underneath the words. The annuity / amortisation formulas are geometric-series sums in disguise.
