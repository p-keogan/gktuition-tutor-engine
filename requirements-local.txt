#     python -m venv .venv-local && . .venv-local/bin/activate
#     pip install -r requirements-local.txt

# package download. No API keys, no Anthropic, no Snowflake.
#
# Spike embedding model: BAAI/bge-small-en-v1.5 (384-dim, L2-normalised),
# served locally via fastembed (ONNX, CPU, ~130MB on disk, no API key).
# NOTE: the *production* embedding choice (hosted vs local) is a separate
# Phase-3 decision — see docs/SNOWFLAKE_EXIT_PLAN.md §3. This pin is the

# ── AGENT_31: hybrid (BM25 + arctic-embed) re-spike (Snowflake-exit Phase-0 v2)─
# v1 (bge-small, vector-only) NO-GO'd on embedding recall. v2 tests the
# architecture Cortex actually uses: a stronger query/document-asymmetric dense
# model from the arctic-embed family + a lexical BM25 index, fused with RRF.
# Both are local, offline, key-free. arctic-embed-m is served via the SAME
# fastembed pin above (no new dep — fastembed 0.3.6 ships the ONNX weights):
#     snowflake/snowflake-arctic-embed-m   (768-dim, one-time download)
# selectable with: build_local_index.py --model snowflake/snowflake-arctic-embed-m
#
# BM25: bm25s — pure-python/numpy BM25, no API key, no network at query time.
# (0.2.7 is yanked upstream; pin a clean release.)
bm25s==0.2.13            # lexical BM25 index for the hybrid retriever
PyStemmer==2.2.0.3       # Snowball English stemmer for BM25 tokenisation

# ── AGENT_33: hosted-librarian decider (Snowflake-exit Phase-0 v4) ───────────

#   reranker  : rerank-2.5 (32K context, instruction-following)
